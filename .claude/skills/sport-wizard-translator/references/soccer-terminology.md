# Swedish Soccer Terminology

This glossary provides domain-specific Swedish translations for soccer/football terms used in Sport Wizard. **Always use these terms** instead of literal translations.

## Core Positions & Roles

| English | Swedish | Notes |
|---------|---------|-------|
| Defender | Back | **NOT** "Försvarare" - "Back" is standard Swedish soccer terminology |
| Midfielder | Mittfält | Used for both singular and role description |
| Attacker / Forward | Forward | Standard Swedish soccer term |
| Goalkeeper | Målvakt | Standard term |
| Substitute | Avbytare | Player not currently on field |
| Captain | Kapten | Team captain |

## Positional Modifiers

Use natural Swedish word order for positions:

| English | Swedish | Pattern |
|---------|---------|---------|
| Left Back | Vänsterback | Position + Role |
| Right Back | Högerback | Position + Role |
| Center Back | Mittback | Position + Role |
| Left Midfielder | Vänster mittfält | Position + Space + Role |
| Right Midfielder | Höger mittfält | Position + Space + Role |
| Left Forward | Vänsterforward | Position + Role |
| Right Forward | Högerforward | Position + Role |

## Important Notes

1. **"Back" not "Försvarare"**: In Swedish soccer context, defenders are called "backar" (plural) or "back" (singular)
2. **"Match" not "Spel"**: Always use "match" for soccer games
3. **Natural compounds**: Swedish often combines words (Vänsterback, Högerforward) without spaces
4. **Position order**: In Swedish, position comes before role (Vänster mittfält, not Mittfält vänster)
